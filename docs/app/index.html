Map Visualization Demo

The original application renders an interactive map using the MapVisualizer library. When running on an ASP.NET server, the DrawMap.ashx handler generates a map image dynamically and supports click-to-query functionality to identify cities at specific coordinates.

How It Works

The map is rendered with a color-coded legend:

Color Value
>95%
90-95%
60-90%
30-60%
<30%
Onbekend

var legend = new Dictionary<Color, string>()

    { ColorTranslator.FromHtml("#FA8E72"), "30-60%" },
    { ColorTranslator.FromHtml("#E15151"), "<30%" },
    { ColorTranslator.FromHtml("#BFBFBF"), "Onbekend" }
};

using (var result = MapHelper.DrawMap(
    new Size(width, height), 50, 50, 50, 50,
    Color.White, new Pen(Color.Black, 1.0f),
    legend, legendCaption, font, legendFontColor, 1.5f))
{
    context.Response.ContentType = "image/png";
    result.Save(context.Response.OutputStream, ImageFormat.Png);
}
